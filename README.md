# 📊 Trader Performance vs Market Sentiment Analysis

## 🚀 Objective
This project analyzes how market sentiment (Fear vs Greed) impacts trader behavior and performance using Hyperliquid trading data.

---

## 📂 Datasets Used
1. Bitcoin Fear & Greed Index
2. Historical Trader Data (Hyperliquid)

---

## 🧹 Data Preparation
- Converted timestamps to daily format
- Merged sentiment and trading datasets on date
- Handled missing values and ensured proper alignment
- Created key metrics:
  - Closed PnL
  - Trade count
  - Position size (USD)
  - Trader segmentation

---

## 📈 Key Analysis

### 1. PnL vs Sentiment
- Fear → high volatility and extreme losses/profits
- Greed → more stable and consistent performance
- Extreme Greed → controlled but profitable behavior

### 2. Trade Frequency
- Fear → highest trading activity (panic behavior)
- Greed → active but more controlled participation
- Neutral → lowest activity

### 3. Position Size
- Fear → very large position spikes (high risk)
- Greed → moderate and consistent sizing

### 4. Trader Segments
- High-frequency traders dominate during Fear
- Low-frequency traders show more stable PnL

---

## 💡 Key Insights
- Fear leads to emotional and high-risk trading behavior
- Greed leads to structured and consistent trading
- Extreme Greed shows disciplined profit-taking

---

## 🧠 Strategy Recommendations
- Reduce leverage and position size during Fear markets
- Increase participation during Greed with controlled risk
- Avoid overtrading during panic conditions
- Focus on consistency rather than aggressive trades

---

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- Seaborn, Matplotlib
- Jupyter Notebook

---

## 📌 Conclusion
Market sentiment strongly influences trader behavior. Understanding these patterns can help design better trading strategies and risk management rules.
